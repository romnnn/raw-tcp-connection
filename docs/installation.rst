============
Installation
============


Stable release
--------------

To install raw_tcp_connection, run this command in your terminal::

    $ pip install raw_tcp_connection

If you don't have `pip`_ installed, see this `guide`_.

.. _pip: https://pip.pypa.io
.. _guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

To install raw_tcp_connection from the sources, first clone the repository::

    $ git clone git://github.com/romnn/raw_tcp_connection

After you obtained a local clone of the source, install with::

    $ python setup.py install
